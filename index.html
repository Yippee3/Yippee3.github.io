Linux从基因组中序列提取

准备染色质定位csv文件，chr1：3284705-3286244，用linux将其改为chr1 3284705 3286244，tab分割，另存为csv，重命名改后缀.csv为.bed，加载bedtools。

bed文件格式为

染色体位置 基因名 得分（直接全部设为1） 正负链（+/-）

  1. BED3: A BED file where each feature is described by chrom, start, and end.

For example: chr1 11873 14409

  1. BED4: A BED file where each feature is described by chrom, start, end, and name.

For example: chr1 11873 14409 uc001aaa.3

  1. BED5: A BED file where each feature is described by chrom, start, end, name, and score.

For example: chr1 11873 14409 uc001aaa.3 0

  1. BED6: A BED file where each feature is described by chrom, start, end, name, score, and strand.

For example: chr1 11873 14409 uc001aaa.3 0 +

Cytoscape

软件窗口

菜单栏

  • File：open（打开一个Cytoscape文件）；New（建立一个新的网络，空的或已经存在的网络）；Import（导入网络数据和属性）；Export（输出数据和图）等
  • Eidt：Undo（撤销）；Redo（重做）；create/destory view (创建/撤销视图)等
  • View：Hide/Show Control Panel(打开或隐藏网络处理板块)；Show Results Panel(网络浏览)等
  • Select：不同点和边选择选项；过滤器等
  • Layout：安排可视化网络，
  • Apps：管理插件（install/update/delete）和添加已经安装的插件
  • Tools:

工具栏

主要为菜单栏file的快捷键：打开、保存、导入(本地、数据库、表格)/导出(网络、表格、图片)以及网络主视图窗口中网络大小调整等

网络处理板块

network：包括所有创建的网络，可以选择相应的网络进行操作
style：属性(node/edge/network)
Node:对点进行设置，包括：点的形状、颜色、大小；点边界线的类型、颜色、宽度；点标签的颜色、大小；点背景色的透明度等
Edge：对边进行设置，包括：边的类型、颜色、宽度；目标处箭头类型等。
Network：对网络整体属性进行设置，包括：背景标题等
Select：面板用于筛选符合特定标准的边

属性浏览版块

查看node/edge/network属性

软件使用

本地输入文件准备

txt或excel文件，至少包括两列，source和target，可以加上相互作用关系

文件导入

File—–Import—–Network from File—–选择文件—–打开—–分别选中列名，标记为source node, target node—–OK

图像调整

左侧Style

插件安装

Apps—–App Manager
